Appendix B: Setting up password-less ssh
========================================
.. Warning (should rewrite for ssh2):

The following instructions describe how to set up **ssh** to allow password-lessauthentication among a collection of machines. These instructions apply to 
ssh1; the process for ssh2 is different. You should first read all the 
instructions before proceeding. Note that these instructions only apply to 
computers running variants of the UNIX operating system.
