Standard Plot Types
-------------------

VisIt comes with eighteen standard plots:
Boundary,
Contour,
Curve,
FilledBoundary,
Histogram,
Label,
Mesh,
``Molecule,``
``MultiCurve,``
ParallelCoordinates,
Pseudocolor,
Scatter,
Spreadsheet,
Streamline,
Subset,
Surface,
Tensor,
Truecolor,
Vector, and Volume. This section explains each plot in detail.
